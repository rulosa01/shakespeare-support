# The Complete Shakespeare — Support

The public support tracker for **The Complete Shakespeare**, an iOS app with every play, all 154 sonnets, and the narrative poems — plus on-device Apple Intelligence to explain any passage.

> A one-person project. Bug reports, feature requests, and general feedback are all welcome here.

## Filing a bug report

[**Open a new issue**](https://github.com/rulosa01/shakespeare-support/issues/new) and please include:

- **iOS version** — Settings → General → About → Software Version
- **iPhone or iPad model**
- **What you were doing** when the issue appeared
- **What you expected to happen**
- **What actually happened**
- **Screenshot** if it's visual

A rough "Hamlet 3.1 looked weird on my iPhone 16 Plus" is more useful than nothing.

## Feature requests

Open an issue and tell me what you'd want. On the radar: bookmarks, highlights, font and theme choices, reading-position resume. If any of those — or anything else — would change how you use the app, I want to hear it.

## About Apple Intelligence

The "Explain" feature uses Apple's on-device Foundation Models framework. It works on iPhone 15 Pro and later, iPads with M-series chips, and any other Apple Intelligence-eligible device, and only when Apple Intelligence is enabled in Settings. On other devices, the Explain UI is hidden — the rest of the reader works the same.

## About this repo

This is **support issues only** — the app's source code isn't published here. If you'd like to discuss the app's approach (FoundationModels, the corpus pipeline, Swift package structure), feel free to open an issue.
